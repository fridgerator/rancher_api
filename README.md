# rancher_api

[Rancher](http://rancher.com/) api V2 for [crystal](https://crystal-lang.org/)

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  rancher_api:
    github: fridgerator/rancher_api
```

## Usage

